# BenTools

Use `/bt` to open the panel.

## Main

- `/bt`
- `/bt help`
- `/bt settings`
- `/reload`

## Auto Sell

- `/bt scan`
- `/bt autosell scan`
- `/bt autosell list`
- `/bt autosell always`
- `/bt autosell never`
- `/bt autosell clear`
- `/bt list`
- `/bt always`
- `/bt never`
- `/bt clear`

Notes:

- Alt-right-click bag item = Always Sell
- Ctrl-Alt-right-click bag item = Never Sell
- open a merchant and use `Sell Marked Items`

## Queue Ringer

- `/bt queue`
- `/bt queue status`
- `/bt queue test`
- `/bt queue on`
- `/bt queue off`
- `/bt queue debug`
- `/bt queue help`

## Mythic+ Finder

- `/bt mplus`
- `/bt mythic`
- `/bt finder`
- `/bt mplus search`
- `/bt mplus status`
- `/bt mplus dump`
- `/bt mplus debug`
- `/bt mplus preset <name>`
